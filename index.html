🔗 Kết nối Google Calendar
📋 Cách lấy Client ID (một lần duy nhất)
1. Vào console.cloud.google.com
2. Tạo project mới → bật Google Calendar API
3. APIs & Services → OAuth consent screen → External
4. Credentials → + Create → OAuth client ID
5. Type: Web application
6. Authorized JS origins: bấm để copy
7. Copy Client ID → dán vào ô dưới
🔒 Lưu trên thiết bị của bạn, không gửi đi đâu cả

📋 Cách export từ Google Calendar
Bước 1. Mở calendar.google.com trên máy tính
Bước 2. Click ⚙️ → Cài đặt
Bước 3. Menu trái → Nhập và xuất
Bước 4. Click Xuất → tải về file .zip
Bước 5. Giải nén → chọn file .ics bên dưới
📂
Chọn file .ics hoặc kéo thả vào đây
Hỗ trợ file từ Google Calendar, Apple Calendar, Outlook
